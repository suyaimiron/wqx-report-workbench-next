# 五强溪水电站水利计算可复核决策工作台

本项目为“水利计算课程设计评优展示”网页成果，用于展示五强溪水电站课程设计的报告复现结果，并提供自定义试算模式。

## GitHub 地址

代码已上传到 GitHub：

[suyaimiron/wqx-report-workbench-next](https://github.com/suyaimiron/wqx-report-workbench-next)

## 运行方式

先进入项目文件夹：

```bash
cd wqx-report-workbench-next
```

安装依赖：

```bash
npm install
```

启动本地网页：

```bash
npm run dev
```

浏览器打开：

```text
http://localhost:3000
```

## 说明

本项目使用 Next.js、TypeScript、Tailwind CSS、Recharts 和 lucide-react 开发。
