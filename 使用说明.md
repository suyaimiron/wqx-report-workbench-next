# 五强溪水电站水利计算工作台使用说明

## 平时打开

双击项目文件夹里的 `启动网站.bat`。

启动后浏览器会打开：

`http://localhost:3000`

关闭命令行窗口即可停止网站。

## 手动运行

```bash
npm install
npm run build
npm run start
```

然后访问：

`http://localhost:3000`

## 修改后更新

如果后续修改了网页代码，重新运行：

```bash
npm run build
npm run start
```

或者删除 `.next` 文件夹后再双击 `启动网站.bat`，它会自动重新构建。
